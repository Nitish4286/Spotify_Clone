# Spotify Clone 🎵

A pixel-perfect, premium Spotify clone built with pure HTML, CSS, and JavaScript — no frameworks, no build tools needed.

## ✨ Features

- 🎵 **Full Music Player** — Play/Pause, Next/Prev, Seek bar, Volume, Shuffle, Repeat
- 📄 **4 Views** — Home, Search (live filter), Library, Playlist Detail
- 💚 **Like Songs** — Save to Liked Songs with toast notifications
- 📋 **Now Playing Panel** — Slide-in panel with queue list
- ⌨️ **Keyboard Shortcuts** — Space, ←/→, S, R, M
- 🎨 **Premium Design** — Dark theme, Spotify green accent, glassmorphism cards
- 📱 **Responsive** — Works on smaller screens too
- 🎭 **Micro-animations** — Equalizer, hover effects, smooth transitions

## 🚀 How to Run

Just open `index.html` in your browser — no server required!

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `→` | Next song |
| `←` | Previous song |
| `S` | Toggle Shuffle |
| `R` | Cycle Repeat mode |
| `M` | Mute / Unmute |

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — SPA router, player state machine, DOM manipulation
- **Web Audio API** — Audio playback support
